# Math Microservices (Node.js + Express)

## 📌 Project Overview
This project is a simple microservice built using Node.js and Express.  
It provides mathematical operations (square and cube) through API endpoints and a user‑friendly frontend form.  
The frontend has **three parallel sections**:
1. Square + Cube together
2. Square only
3. Cube only

The goal of this assignment was to learn the complete lifecycle of a microservice: setup, coding, running locally, pushing to GitHub, deploying, and documenting with screenshots.

---

## 🛠️ Technology Stack
- **Language:** JavaScript (Node.js)
- **Framework:** Express.js
- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git + GitHub
- **Deployment Options:** Docker / Heroku / Render (free tiers)

## 🚀 Project Setup
1. Create project folder:
   ```bash
   mkdir math-microservices
   cd math-microservices
2.  Initialize Node.js project:
    npm init -y
3.  Install dependencies:
    npm install express
4.  Create files:
- server.js → backend code
- public/index.html → frontend form
- README.md → documentation
- Dockerfile → optional for container deployment
